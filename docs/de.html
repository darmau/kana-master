Funktionen

振

Furigana-Annotation

Alt+Klick auf beliebigen japanischen Text, um sofort Furigana-Lesungen (Ruby) über den Kanji anzuzeigen.

訳

Streaming-Übersetzung

Echtzeit-Streaming-Übersetzung in 17 Sprachen. Die Übersetzung erscheint Wort für Wort.

読

Lesemodus

Ablenkungsfreie Leseansicht mit Absatzauswahl, Stapelübersetzung und sequenzieller TTS-Wiedergabe.

音

Text-zu-Sprache

Natürliche japanische Aussprache über OpenAI TTS mit progressivem Vorladen.

Unterstützte API-Anbieter

Verwenden Sie Ihren eigenen API-Schlüssel. Jede Funktion (Furigana, Übersetzung, Grammatik, TTS) kann einen anderen Anbieter und ein anderes Modell verwenden.

OpenAI Anthropic Google Gemini Chrome integrierte KI

Anleitung

1

Installieren und konfigurieren

Laden Sie die Erweiterung in Chrome, öffnen Sie die Optionsseite und geben Sie Ihre API-Schlüssel ein. Wählen Sie das Modell für jede Funktion.

2

Alt+Klick zum Annotieren

Halten Sie Alt gedrückt und klicken Sie auf japanischen Text auf einer Webseite. Furigana erscheinen über den Kanji und eine Streaming-Übersetzung wird darunter angezeigt.

3

Lesemodus

Klicken Sie auf das Erweiterungssymbol und öffnen Sie den Lesemodus für ein sauberes Leseerlebnis. Wählen Sie Absätze, übersetzen Sie stapelweise und hören Sie TTS.
