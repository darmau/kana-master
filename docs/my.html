ဂျပန်ဘာသာ လေ့လာသောအခါ ကန်ဂျီ ဖတ်နည်းက ခက်ခဲသည်။

အင်္ဂါရပ်များ

振

Furigana မှတ်ချက်

မည်သည့် ဂျပန်စာသားကိုမဆို Alt+Click နှိပ်ပါ၊ ကန်ဂျီအပေါ်တွင် furigana (ruby) အသံထွက် ချက်ချင်းပေါ်လာပါမည်။

訳

Streaming ဘာသာပြန်

ဘာသာ ၁၈ မျိုးသို့ အချိန်နှင့်တစ်ပြေးညီ streaming ဘာသာပြန်ချက်။ ဘာသာပြန်ချက် စကားလုံးတစ်လုံးချင်း ပေါ်လာပါသည်။

文

သဒ္ဒါ ခွဲခြမ်းစိတ်ဖြာခြင်း

မည်သည့်ဝါကျကိုမဆို အသေးစိတ် သဒ္ဒါခွဲခြမ်းစိတ်ဖြာခြင်း ရယူပါ။ ကြိယာပုံစံ၊ သဒ္ဒါပုံစံ နှင့် ဝါကျဖွဲ့စည်းပုံကို ရှင်းပြသည်။

読

Reader Mode

အနှောင့်အယှက်ကင်းသော ဖတ်ရှုရေးမြင်ကွင်း၊ စာပိုဒ်ရွေးချယ်ခြင်း၊ အစုလိုက်ဘာသာပြန်ခြင်းနှင့် TTS ဖွင့်ခြင်း ပါဝင်သည်။

音

စာသားမှ အသံ

OpenAI / Google TTS ဖြင့် သဘာဝကျသော ဂျပန်အသံထွက်၊ တဖြည်းဖြည်း ကြိုတင်ရယူခြင်းကို ပံ့ပိုးသည်။

単

ဝေါဟာရ စုဆောင်းခြင်း

ဘာသာပြန်ချက်ရှိ စကားလုံးကို နှိပ်၍ အဖတ်၊ အဓိပ္ပာယ်ဖွင့်ဆိုချက် နှင့် ဝါကျအကြောင်းအရာနှင့်တကွ ဝေါဟာရစာရင်းတွင် သိမ်းဆည်းပါ။

問

စာဖတ်ပြီး စစ်ဆေးခြင်း

သင်ဖတ်သော အကြောင်းအရာအပေါ် အခြေခံ၍ နားလည်မှု စစ်ဆေးမေးခွန်းများ ဖန်တီးပါ။ JLPT အဆင့်အလိုက် ခက်ခဲမှု ချိန်ညှိပြီး တိုးတက်မှု ခြေရာခံသည်။

ပံ့ပိုးသော API ဝန်ဆောင်မှုများ

သင့်ကိုယ်ပိုင် API key ကို အသုံးပြုပါ။ လုပ်ဆောင်ချက်တစ်ခုစီ (furigana, ဘာသာပြန်, သဒ္ဒါ, TTS) အတွက် မတူညီသော ဝန်ဆောင်မှုနှင့် model ကို ရွေးချယ်နိုင်ပါသည်။

OpenAI Anthropic Google Gemini Chrome Built-in AI

အသုံးပြုနည်း

1

ထည့်သွင်းခြင်းနှင့် ပြင်ဆင်ခြင်း

Chrome တွင် extension ကို load လုပ်ပါ၊ ရွေးချယ်စရာ စာမျက်နှာကိုဖွင့်ပြီး သင့် API key ကို ထည့်သွင်းပါ။ လုပ်ဆောင်ချက်တစ်ခုစီအတွက် model ကို ရွေးပါ။

2

Alt+Click နှိပ်၍ မှတ်ချက်ထည့်ခြင်း

Alt ခလုတ်ကို ဖိထားပြီး ဝဘ်စာမျက်နှာပေါ်ရှိ ဂျပန်စာသားကို click နှိပ်ပါ။ ကန်ဂျီအပေါ်တွင် furigana ပေါ်လာပြီး အောက်တွင် streaming ဘာသာပြန်ချက် ပြသပါမည်။

3

Reader Mode

Extension icon ကို click နှိပ်ပြီး Reader Mode ကိုဖွင့်ပါ။ စာပိုဒ်များ ရွေးချယ်ခြင်း၊ အစုလိုက် ဘာသာပြန်ခြင်းနှင့် TTS နားထောင်ခြင်းတို့ ပြုလုပ်နိုင်ပါသည်။
