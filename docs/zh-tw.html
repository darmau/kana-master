功能特色

振

振假名標註

Alt+點擊任意日文文字，即可在漢字上方添加假名讀音（ruby 注音）。

訳

串流翻譯

支援 17 種目標語言的即時串流翻譯，譯文逐字逐句呈現。

読

閱讀器模式

無干擾的閱讀介面，支援段落選擇、批次翻譯和順序朗讀。

音

文字轉語音

基於 OpenAI TTS 的自然日語發音，支援漸進式預載。

支援的 API 服務商

自備 API 金鑰，每個功能（振假名、翻譯、文法、TTS）可以使用不同的服務商和模型。

OpenAI Anthropic Google Gemini Chrome 內建 AI

使用方法

1

安裝與設定

在 Chrome 中載入擴充功能，開啟設定頁面，輸入您的 API 金鑰。為每個功能選擇要使用的模型。

2

Alt+點擊標註

按住 Alt 鍵並點擊網頁上的日文文字。漢字上方會出現假名讀音，下方會顯示串流翻譯。

3

閱讀器模式

點擊擴充功能圖示並開啟閱讀器模式，獲得乾淨的閱讀體驗。選擇段落、批次翻譯、聆聽 TTS 朗讀。
