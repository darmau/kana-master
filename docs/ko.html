데모

日本語にほんごを勉強べんきょうする時とき、漢字かんじの読よみ方かたが難むずかしい。
일본어를 공부할 때, 한자 읽는 법이 어렵다.

기능

振

후리가나 표기

일본어 텍스트를 Alt+클릭하면 한자 위에 후리가나(루비) 읽기가 즉시 표시됩니다.

訳

스트리밍 번역

17개 언어로의 실시간 스트리밍 번역. 번역이 단어별로 순차적으로 표시됩니다.

読

리더 모드

방해 없는 읽기 화면에서 단락 선택, 일괄 번역, 순차 TTS 재생을 지원합니다.

音

텍스트 음성 변환

OpenAI TTS 기반의 자연스러운 일본어 발음, 점진적 프리페칭을 지원합니다.

지원 API 제공업체

본인의 API 키를 사용하세요. 각 기능(후리가나, 번역, 문법, TTS)마다 다른 제공업체와 모델을 선택할 수 있습니다.

OpenAI Anthropic Google Gemini Chrome 내장 AI

사용 방법

1

설치 및 설정

Chrome에서 확장 프로그램을 로드하고, 설정 페이지를 열어 API 키를 입력하세요. 각 기능에 사용할 모델을 선택합니다.

2

Alt+클릭으로 표기

Alt 키를 누른 채 웹페이지의 일본어 텍스트를 클릭하세요. 한자 위에 후리가나가 나타나고, 아래에 스트리밍 번역이 표시됩니다.

3

리더 모드

확장 프로그램 아이콘을 클릭하고 리더 모드를 열어 깔끔한 읽기 환경을 즐기세요. 단락을 선택하고, 일괄 번역하고, TTS로 들을 수 있습니다.
