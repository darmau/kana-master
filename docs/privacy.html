Privacy Policy

Kana Master Chrome Extension

Last updated: March 23, 2026

Summary: Kana Master does not collect, store, or transmit any personal data to us. All data processing happens locally in your browser or is sent directly to the third-party API providers you choose to configure.

What Data We Collect

Kana Master does not collect any personal information. We do not operate any servers, and no data is sent to us at any time.

Data Stored Locally

The following data is stored locally on your device using Chrome's built-in storage APIs (chrome.storage.sync and chrome.storage.local):

  • API Keys — Your API keys for OpenAI, Anthropic, and/or Google, which you voluntarily enter in the extension's options page. These keys are stored in your browser's synced storage and are never sent anywhere other than the respective API provider's endpoint.
  • User Preferences — Your selected models, target language, TTS voice, translation engine, and other settings.
  • Reader Content — When you use Reader Mode, the extracted page content is temporarily stored in chrome.storage.local to pass it to the reader tab. This data is not persisted beyond the session.

Data Sent to Third Parties

When you use Kana Master's features, the text you select or process is sent to the API provider(s) you have configured:

  • OpenAI API (api.openai.com or your custom base URL) — for furigana annotation, translation, grammar analysis, and text-to-speech.
  • Anthropic API (api.anthropic.com) — for furigana annotation, translation, and grammar analysis.
  • Google Generative Language API (generativelanguage.googleapis.com) — for furigana annotation, translation, and grammar analysis.

Data is sent only to providers for which you have entered an API key and selected a model. The extension does not send data to any provider you have not configured. Please refer to each provider's own privacy policy for how they handle your data:

  • OpenAI Privacy Policy
  • Anthropic Privacy Policy
  • Google Privacy Policy

Chrome Built-in AI

If you choose the "Local" translation engine, translations are performed entirely on your device using Chrome's Built-in AI Translator API. No data leaves your browser in this mode.

Permissions

The extension requests the following browser permissions:

  • storage — to save your settings and API keys locally.
  • activeTab — to interact with the current tab when you activate the extension.
  • Host permissions — to make API requests to the configured providers and to inject the content script on web pages.

Analytics and Tracking

Kana Master does not include any analytics, telemetry, tracking scripts, or advertising. There are no cookies set by the extension.

Data Security

Your API keys are stored using Chrome's chrome.storage.sync API, which encrypts data at rest when you are signed into Chrome. API requests are made over HTTPS. We recommend keeping your API keys confidential and rotating them periodically.

Children's Privacy

Kana Master does not knowingly collect any information from anyone, including children under the age of 13.

Changes to This Policy

If this privacy policy is updated, the changes will be posted on this page with an updated "Last updated" date.

Contact

If you have questions about this privacy policy, please open an issue on the GitHub repository.
