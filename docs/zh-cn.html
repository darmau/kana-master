Kana Master

为网页上的日文添加振假名（furigana）和多语言翻译的 Chrome 扩展

Chrome 扩展 · Manifest V3 · 开源

English 简体中文 繁體中文 한국어 Tiếng Việt नेपाली မြန်မာ

演示

日本語にほんごを勉強べんきょうする時とき、漢字かんじの読よみ方かたが難むずかしい。
学习日语时，汉字的读法很难。

功能特性

振

振假名标注

Alt+点击任意日文文本，即可在汉字上方添加假名读音（ruby 注音）。

訳

流式翻译

支持 17 种目标语言的实时流式翻译，译文逐字逐句呈现。

読

阅读器模式

无干扰的阅读界面，支持段落选择、批量翻译和顺序朗读。

音

文本转语音

基于 OpenAI TTS 的自然日语发音，支持渐进式预加载。

支持的 API 服务商

自带 API 密钥，每个功能（振假名、翻译、语法、TTS）可以使用不同的服务商和模型。

OpenAI Anthropic Google Gemini Chrome 内置 AI

使用方法

1

安装和配置

在 Chrome 中加载扩展，打开设置页面，输入你的 API 密钥。为每个功能选择要使用的模型。

2

Alt+点击标注

按住 Alt 键并点击网页上的日文文本。汉字上方会出现假名读音，下方会显示流式翻译。

3

阅读器模式

点击扩展图标并打开阅读器模式，获得干净的阅读体验。选择段落、批量翻译、收听 TTS 朗读。

支持的语言

可将日语翻译为以下任一目标语言：

隐私政策

Kana Master 是开源项目。你的 API 密钥仅存储在本地，不会发送到你配置的 API 服务商以外的任何服务器。
