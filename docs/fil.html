Mga Tampok

振

Furigana Annotation

Alt+Click sa anumang tekstong Hapon para agad na magdagdag ng furigana (ruby) na pagbasa sa ibabaw ng mga kanji.

訳

Streaming na Pagsasalin

Real-time streaming na pagsasalin sa 17 wika. Lumalabas ang salin nang salita-salita habang ginagawa ito.

読

Reader Mode

Walang-abala na reading view na may pagpili ng talata, batch na pagsasalin, at sunud-sunod na TTS playback.

音

Text-to-Speech

Natural na pagbigkas ng Hapon sa pamamagitan ng OpenAI TTS, na may progressive prefetching.

Mga Suportadong API Provider

Gamitin ang sarili mong API key. Bawat function (furigana, pagsasalin, grammar, TTS) ay maaaring gumamit ng ibang provider at modelo.

OpenAI Anthropic Google Gemini Chrome Built-in AI

Paano Gamitin

1

I-install at I-configure

I-load ang extension sa Chrome, buksan ang options page, at ilagay ang iyong mga API key. Piliin kung anong modelo ang gagamitin para sa bawat function.

2

Alt+Click para Mag-annotate

Pindutin ang Alt at i-click ang anumang tekstong Hapon sa webpage. Lalabas ang furigana sa ibabaw ng kanji at streaming na salin sa ibaba.

3

Reader Mode

I-click ang extension icon at buksan ang Reader Mode para sa malinis na karanasan sa pagbasa. Pumili ng mga talata, magsalin nang sabay-sabay, at makinig sa TTS.

Patakaran sa Privacy

Ang Kana Master ay open source. Ang iyong mga API key ay lokal lang na naka-store at hindi kailanman ipapadala sa anumang server maliban sa mga API provider na iyong na-configure.
