Features

振

Furigana Annotation

Alt+Click on any Japanese text to instantly add furigana (ruby) readings above kanji characters.

訳

Streaming Translation

Real-time streaming translation into 18 languages. Translation appears word by word as it's generated.

文

Grammar Analysis

Get detailed grammar breakdowns for any sentence. Conjugation forms, grammar patterns, and sentence structure explained.

読

Reader Mode

Distraction-free reading view with paragraph selection, bulk translation, and sequential TTS playback.

音

Text-to-Speech

Listen to natural Japanese pronunciation powered by OpenAI or Google TTS, with progressive prefetching.

単

Vocabulary Collection

Click any word in translations to save it to your vocabulary list with readings, definitions, and sentence context.

問

Reading Quizzes

Generate comprehension quizzes based on what you read. Difficulty adapts to your JLPT level, with progress tracking.

Supported API Providers

Bring your own API key. Each function (furigana, translation, grammar, TTS) can use a different provider and model.

OpenAI Anthropic Google Gemini Chrome Built-in AI

How to Use

1

Install & Configure

Load the extension in Chrome, open the options page, and enter your API key(s). Choose which model to use for each function.

2

Alt+Click to Annotate

Hold Alt and click on any Japanese text on a webpage. Furigana readings will appear above kanji, and a streaming translation will display below.

3

Reader Mode

Click the extension icon and open Reader Mode for a clean reading experience. Select paragraphs, translate in bulk, and listen to the text with TTS.
