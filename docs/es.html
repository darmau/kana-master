Demostración

日本語にほんごを勉強べんきょうする時とき、漢字かんじの読よみ方かたが難むずかしい。
Al estudiar japonés, la lectura de los kanji es difícil.

Características

振

Anotación Furigana

Alt+Clic en cualquier texto japonés para agregar instantáneamente lecturas furigana (ruby) sobre los kanji.

訳

Traducción en streaming

Traducción en tiempo real a 17 idiomas. La traducción aparece palabra por palabra a medida que se genera.

読

Modo lector

Vista de lectura sin distracciones con selección de párrafos, traducción por lotes y reproducción TTS secuencial.

音

Texto a voz

Pronunciación japonesa natural con OpenAI TTS, con precarga progresiva.

Proveedores de API compatibles

Usa tu propia clave API. Cada función (furigana, traducción, gramática, TTS) puede usar un proveedor y modelo diferente.

OpenAI Anthropic Google Gemini Chrome IA integrada

Cómo usar

1

Instalar y configurar

Carga la extensión en Chrome, abre la página de opciones e ingresa tus claves API. Elige qué modelo usar para cada función.

2

Alt+Clic para anotar

Mantén Alt y haz clic en cualquier texto japonés en una página web. Los furigana aparecen sobre los kanji y una traducción en streaming se muestra debajo.

3

Modo lector

Haz clic en el icono de la extensión y abre el modo lector para una experiencia de lectura limpia. Selecciona párrafos, traduce por lotes y escucha con TTS.

Política de privacidad

Kana Master es de código abierto. Tus claves API se almacenan localmente y nunca se envían a ningún servidor que no sea el de los proveedores de API que configures.
