عند دراسة اللغة اليابانية، تكون قراءة الكانجي صعبة.

الميزات

振

تعليق فوريغانا

Alt+نقر على أي نص ياباني لإضافة قراءات فوريغانا (روبي) فورياً فوق أحرف الكانجي.

訳

ترجمة متدفقة

ترجمة فورية متدفقة إلى 18 لغة. تظهر الترجمة كلمة بكلمة أثناء إنشائها.

文

تحليل القواعد

احصل على تحليل نحوي مفصل لأي جملة. شرح تصريفات الأفعال وأنماط القواعد وبنية الجملة.

読

وضع القارئ

واجهة قراءة خالية من المشتتات مع اختيار الفقرات والترجمة الجماعية وتشغيل TTS المتسلسل.

音

تحويل النص إلى كلام

نطق ياباني طبيعي عبر OpenAI / Google TTS مع جلب مسبق تدريجي.

単

جمع المفردات

انقر على أي كلمة في الترجمات لحفظها في قائمة المفردات مع القراءات والتعريفات وسياق الجملة.

問

اختبارات القراءة

أنشئ اختبارات فهم بناءً على ما تقرأه. تتكيف الصعوبة مع مستوى JLPT الخاص بك، مع تتبع التقدم.

مزودو API المدعومون

استخدم مفتاح API الخاص بك. كل وظيفة (فوريغانا، ترجمة، قواعد، TTS) يمكنها استخدام مزود ونموذج مختلف.

OpenAI Anthropic Google Gemini Chrome AI مدمج

كيفية الاستخدام

1

التثبيت والإعداد

قم بتحميل الإضافة في كروم، افتح صفحة الخيارات وأدخل مفاتيح API الخاصة بك. اختر النموذج المراد استخدامه لكل وظيفة.

2

Alt+نقر للتعليق

اضغط مع الاستمرار على Alt وانقر على أي نص ياباني على صفحة ويب. ستظهر الفوريغانا فوق الكانجي وترجمة متدفقة أسفله.

3

وضع القارئ

انقر على أيقونة الإضافة وافتح وضع القارئ لتجربة قراءة نظيفة. اختر الفقرات وترجم بالجملة واستمع عبر TTS.
